## 🎯 Bot Commands

### 📖 စာအုပ် ရှာဖွေ

| Command | အဓိပ္ပာယ် |
|---------|-----------|
| `/start` | Bot စတင်ရန် |
| `/authors` | စာရေးသူများ စာရင်း |
| `/search <keyword>` | စာအုပ်ရှာဖွေရန် |
| `/refresh` | ဒေတာပြန်ဖတ်ရန် |
| `/stats` | စာရင်းဇယား |

### 📝 Book Notes

| Command | အဓိပ္ပာယ် |
|---------|-----------|
| `/addnote စာအုပ် - 5` | Note ထည့်ရန် (ပြီးရင် reply မှာ note ရေးပါ) |
| `/note စာအုပ်နာမည်` | Note ကြည့်ရန် |
| `/mynote` | ကိုယ်ရေးထားသည့် Note များ |
| `/delnote <id>` | Note ဖျက်ရန် |

### ⚙️ Admin Commands

| Command | အဓိပ္ပာယ် |
|---------|-----------|
| `/add စာရေးသူ - စာအုပ် - link` | စာအုပ်အသစ်ထည့်ရန် (Admin သာ) |
| `/del စာရေးသူ - စာအုပ် - link` | စာအုပ်ဖျက်ရန် (Admin သာ) |
| `/ban` | Ban ရန် (message reply) |
| `/unban @username` | Unban ရန် |
| `/setwelcome မင်္ဂလာပါ {name}` | Welcome message ပြင်ရန် |

## 🎯 Book Notes အသေးစိတ်

### Note ထည့်နည်း

1. `/addnote မီဆိုဟင်းချို 5` ရိုက်ပါ
2. Bot က reply prompt ပြပါမည်
3. Reply မှာ note content ရေးပါ
4. Note ကို GitHub ထဲ သိမ်းထားပါမည်

### Note ကြည့်နည်း

- `/note မီဆိုဟင်းချို` → ရေးထားသူတိုင်းရဲ့ note များ ပေါ်လာပါမည်
- Note များများရှိရင် Button များနဲ့ ရွေးဖတ်နိုင်ပါသည်
- ⭐ rating ပါပါမည်

### Note ဖျက်နည်း

- `/delnote <note_id>` → ကိုယ်ရေးထားသည့် note သာ ဖျက်နိုင်ပါသည်

## 🚀 Railway ဖြင့် Deploy နည်း (Free)

### အဆင့် ၁ — GitHub Account ဖွင့်ပါ

[github.com](https://github.com) မှာ Account ဖွင့်ပါ။ ရှိပြီးသားဆိုရင် ကျော်လို့ရပါတယ်။

### အဆင့် ၂ — Bot Token ယူပါ

1. Telegram ထဲမှာ **@BotFather** ကို ရှာပါ
2. `/newbot` ရိုက်ပါ
3. Bot နာမည် ထည့်ပါ (ဥပမာ - `Saroatsin Bot`)
4. Username ထည့်ပါ (ဥပမာ - `my_saroatsin_bot`)
5. BotFather က **Token** ပေးပါမည် — `8644464116:AA...` ပုံစံ
6. ဒီ Token ကို **ကောင်းကောင်း ကူးထားပါ**

### အဆင့် ၃ — GitHub Personal Access Token (GITHUB_TOKEN) ယူပါ

စာအုပ်ထည့်/ဖျက်နိုင်ဖို့နဲ့ Note သိမ်းနိုင်ဖို့ GITHUB_TOKEN လိုပါမည်။

1. [github.com/settings/tokens](https://github.com/settings/tokens) သွားပါ
2. **Generate new token (classic)** နှိပ်ပါ
3. **Note** မှာ `saroatsin-bot` ထည့်ပါ
4. **Expiration** မှာ `90 days` ရွေးပါ
5. **Scopes** ထဲမှာ `repo` ကို tick ပေးပါ
6. **Generate token** နှိပ်ပါ
7. Token ကို **ကောင်းကောင်း ကူးထားပါ**

### အဆင့် ၄ — Railway Account ဖွင့်ပါ

1. [railway.app](https://railway.app) ကို သွားပါ
2. **Sign In with GitHub** နှိပ်ပါ
3. GitHub Account နဲ့ ချိတ်ပါ

### အဆင့် ၅ — Project ဖန်တီးပါ

1. **New Project** နှိပ်ပါ
2. **Deploy from GitHub Repo** ရွေးပါ
3. `whispermmepub/saroatsin-bot` repo ကို ရွေးပါ

### အဆင့် ၆ — Environment Variable ထည့်ပါ

1. Project ထဲ ဝင်ပါ
2. **Variables** tab ကို နှိပ်ပါ
3. **New Variable** နှိပ်ပါ
4. ထည့်ရန် —

| Variable | Value |
|----------|-------|
| `TELEGRAM_BOT_TOKEN` | အဆင့် ၂ မှာ ယူထားတဲ့ Bot Token |
| `GITHUB_TOKEN` | အဆင့် ၃ မှာ ယူထားတဲ့ Personal Access Token |

### အဆင့် ၇ — Deploy ဖြစ်ပါစေ

Variable ထည့်ပြီးသည်နှင့် Railway က **အလိုအလျောက်** Deploy စတင်ပါမည်။

- ✅ `Building` → `Deploying` → **`Active`** ဖြစ်ရင် Bot အဆင်ပြေပါပြီ
- 🟢 **24/7 အဆက်မပြတ် Run** ပါမည်
- 📱 ဖုန်း Screen Off ဖြစ်သည်ဖြစ်ပါစေ Bot ဆက်အလုပ်လုပ်ပါမည်

## 💻 Local Run (Termux / PC)

```bash
# Requirements install
pip install -r requirements.txt

# Bot Token ထည့်ပါ
export TELEGRAM_BOT_TOKEN="your-bot-token-here"
export GITHUB_TOKEN="your-github-token-here"

# Bot Run
python bot.py
```

### Termux မှာ 24/7 Run နည်း

```bash
# Wake lock - Screen off ဖြစ်လည်း run ဆက်ရန်
termux-wake-lock

# Background run
nohup python bot.py &
```
